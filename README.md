# 🌍 Global Development Data Analysis & Visualization

## 📌 Overview
This project analyzes global development data using the Gapminder dataset. It focuses on understanding the relationship between GDP per capita, life expectancy, and population across countries and over time using interactive visualizations.

## 🛠️ Tools & Technologies
- Python  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Plotly (Interactive Visualizations)  

## 📂 Project Structure
- notebook/ → Jupyter Notebook with full analysis  
- reports/ → Exported visualizations

## 📊 Key Insights
- Higher GDP per capita is strongly associated with higher life expectancy  
- Significant improvements in life expectancy over time across most countries  
- Population size plays a major role in global development trends  
- Clear disparities exist between continents in both GDP and life expectancy  

## 📈 Visualizations Included
- Bar charts (country-level trends)  
- Scatter plots (GDP vs Life Expectancy)  
- Bubble charts (population impact)  
- Facet plots (continent-wise comparison)  
- Animated plots (year-wise progression)  
- Choropleth maps (global life expectancy distribution)  

## 🎥 Key Features
- Interactive bubble charts with animation over time  
- Log-scale analysis for better distribution understanding  
- Geographic visualization using world maps  
- Multi-dimensional analysis (GDP, life expectancy, population)
